    \section{Conclusiones del estudio}\label{conclusiones-del-estudio}

En esta secci√≥n final de nuestro estudio, vamos a \textbf{evaluar
cuantitativamente el ajuste de nuestros modelos} farmacocin√©ticos a los
datos experimentales obtenidos de ratones tratados con sunitinib. Para
ello, utilizaremos dos m√©tricas muy importantes:

\begin{itemize}
\item
  \textbf{SSR (Suma de los Cuadrados del Residuo)}: mide el error total
  entre los valores predichos por un modelo y los valores observados.
  Cuanto menor sea el SSR, mejor es el ajuste del modelo.
\item
  \textbf{AIC (Criterio de Informaci√≥n de Akaike)}: eval√∫a la calidad de
  un modelo teniendo en cuenta tanto el ajuste a los datos como la
  complejidad del modelo (n√∫mero de par√°metros). Un AIC m√°s bajo indica
  un modelo m√°s eficiente.
\end{itemize}

Vamos a comparar \textbf{cuatro funciones modelo} y determinar cu√°l se
ajusta mejor a nuestros datos reales.

    \subsection{C√°lculo del SSR}\label{cuxe1lculo-del-ssr}

El SSR (Suma de los Cuadrados del Residuo) se calcula usando la
siguiente f√≥rmula:

\[
SSR = \sum_{i=1}^{n} (y_i - \hat{y}_i)^2
\]

Donde:

\begin{itemize}
\tightlist
\item
  \(y_i\) son los valores reales.
\item
  \(\hat{y}_i\) son los valores predichos por el modelo.
\end{itemize}

    \subsection{C√°lculo del AIC}\label{cuxe1lculo-del-aic}

El AIC (Criterio de Informaci√≥n de Akaike) penaliza la complejidad del
modelo. La f√≥rmula que usaremos es:

\[
AIC = n \cdot \ln\left(\frac{SSR}{n}\right) + 2k
\]

Donde:

\begin{itemize}
\tightlist
\item
  \(n\): n√∫mero de observaciones.
\item
  \(SSR\): suma de los cuadrados de los residuos.
\item
  \(k\): n√∫mero de par√°metros del modelo.
\end{itemize}

Asumiremos lo siguiente para nuestros modelos: - Modelo 1: 2 par√°metros
- Modelo 2: 3 par√°metros - Modelo 3: 2 par√°metros - Modelo 4: 4
par√°metros

    \subsubsection{Test de Yuan para comparar
AUC\_inf}\label{test-de-yuan-para-comparar-auc_inf}

El estad√≠stico de prueba (Z) se calcula como:

\[

\]

El valor p (p-value) se calcula como:

\[
p\text{-value} = 2 \times \text{P}(Z > |z|)
\]
